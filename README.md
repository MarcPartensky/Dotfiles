# My dotfiles config.

## Run with Docker
```
docker run -it marcpartensky/dotfiles
```

## Zsh plugins
* zsh autocomplete

## Aliases
* dotm => ./main.sh
* dotf => ./functions.sh
* dota => ./aliases.sh
* dote => ./exports.sh
* dotz => ~/.zshenv
* dott => ~/.tmux.conf
* dotg => ~/.gitconfig
* doti => ~/.gitignore_global
